# Machine Learning Practice Repository

A repository for learning and practicing Machine Learning concepts and projects.

## Contents

### Notebooks & Practice

- **data_gathering/** - Data collection and processing notebooks
  - `api.ipynb` - Fetching data from APIs and creating dataframes
  - `csv_work.ipynb` - CSV data manipulation and processing
  - `json_sql.ipynb` - JSON and SQL data loading and processing

- **practice_project/** - End-to-end ML project with practical implementation
  - `e2e_project.ipynb` - Complete project demonstrating ML workflow

- **online_machine_learning/** - Online ML learning materials
  - `online_ml.ipynb` - Online learning algorithms and concepts

### Data

- **data/** - Datasets used while learning
  - `practice_project/` - Project-specific datasets
    - `placement.csv` - Sample dataset
  - `data_gathering/` - Data processing datasets
    - `BX-Books.csv` - Book dataset
    - `IPL Matches 2008-2020.csv` - IPL cricket matches data
    - `aug_train.csv` - Training dataset
    - `movies.csv` - Movie dataset
    - `movie_titles_metadata.tsv` - Movie metadata (TSV format)
    - `test.csv` - Test dataset
    - `train.json` - JSON training data
    - `world.sql` - SQL database file
    - `zomato.csv` - Zomato restaurant data

## Purpose

This repository serves as a learning hub where I practice and commit Machine Learning implementations, experiments, and learnings. Each commit represents a step in the learning journey.

## Technologies

- Python
- Jupyter Notebooks
- Machine Learning libraries (scikit-learn, pandas, numpy, etc.)
- API data collection
- JSON and SQL data processing
